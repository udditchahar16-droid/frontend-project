# 🚖 Ride Booking Web App (Uber/Rapido Clone)

A professional front-end ride booking web application built using **HTML, CSS, and JavaScript**. This project simulates core features of apps like Uber and Rapido, including ride booking, fare calculation, and ride history.

---

## 📌 Features

- 🔐 Login & Signup (with basic validation)
- 📍 Book Ride (Pickup & Drop location)
- 💰 Fare Calculation (based on distance logic)
- 🚗 Ride Status Simulation (Requested → Accepted → Completed)
- 📜 Ride History (stored using LocalStorage)
- 👨‍✈️ Driver Panel UI (accept/reject rides)
- 📱 Fully Responsive Design (Mobile + Desktop)
- 🎨 Modern UI with smooth animations

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)
- LocalStorage (for data persistence)
